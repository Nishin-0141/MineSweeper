# フロー
| 手順 | コマンド例 | 内容 |
| -- | -- | -- |
| 1 | npm run dev | Next.js 起動（http://localhost:3000） |
| 2 | go run main.go | Go API 起動（http://localhost:8080）|
| 3 | .env.local 設定 | Next.js → Go API に接続 |
| 4 | DynamoDB Local | 完全ローカルでランキング動作 |

# ① フロントエンド起動
```sh
sh frontend-start.sh
```

# ① バックエンド起動
```sh
sh backend-start.sh
```

# ④ DynamoDB のローカル運用
```
docker run -p 8000:8000 amazon/dynamodb-local
```

URL:
```
http://localhost:8000
```
テーブル作成例
```sh
aws dynamodb create-table \
  --table-name MinesweeperRanking \
  --attribute-definitions AttributeName=UserId,AttributeType=S \
  --key-schema AttributeName=UserId,KeyType=HASH \
  --billing-mode PAY_PER_REQUEST \
  --endpoint-url http://localhost:8000
```

Go での接続コード：
```go
cfg, err := config.LoadDefaultConfig(
    context.TODO(),
    config.WithEndpointResolver(
        aws.EndpointResolverFunc(func(service, region string) (aws.Endpoint, error) {
            return aws.Endpoint{URL: "http://localhost:8000"}, nil
        }),
    ),
)
```
